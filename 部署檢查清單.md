## 🚀 快速部署步驟

### ⏱️ 預計時間：5 分鐘

1. **建立 GitHub 帳戶**（如無）
   - 訪問 github.com
   - 點擊 Sign up
   - 完成註冊

2. **建立 Repository**
   - 名稱：`pengmei-dumpling`
   - 勾選 Public
   - 勾選 Add a README file

3. **上傳檔案**
   - 點擊 Add file → Upload files
   - 選擇上述所有檔案
   - Commit changes

4. **啟用 Pages**
   - Settings → Pages
   - Branch: main
   - Folder: / (root)
   - Save

5. **等待部署**
   - 1-2 分鐘後會收到通知
   - 複製生成的網址

6. **分享給別人**
   - 複製網址：
   ```
   https://[用戶名].github.io/pengmei-dumpling
   ```
