# 🚀 快速開始 - 啟動本地伺服器

現在您的網站可以在 **http://localhost:8000** 訪問！

---

## 📋 方案選擇

### ✅ 推薦方案：Python 伺服器（最簡單）

#### **Windows 用戶**
1. 雙擊 `啟動伺服器.bat` 檔案
2. 在瀏覽器打開 `http://localhost:8000`

#### **Mac / Linux 用戶**
```bash
python3 server.py
```

然後在瀏覽器打開 `http://localhost:8000`

---

### 📦 替代方案：Node.js 伺服器

如果您已安裝 Node.js：

```bash
node server.js
```

然後在瀏覽器打開 `http://localhost:8000`

---

## 🌐 訪問網站

啟動伺服器後，在瀏覽器打開：

```
http://localhost:8000
```

您將看到膨嬤水餃 & 小帥甜點店的完整網站！

---

## 🛑 停止伺服器

在命令行窗口按 **`Ctrl + C`** 即可停止伺服器

---

## ⚙️ 技術細節

### Python 伺服器
- 使用 Python 內置的 `http.server`
- 自動防止快取（便於開發）
- 自動將 `/` 重定向至 `index.html`

### Node.js 伺服器
- 使用 Node.js 內置的 `http` 模組
- 支援所有常見 MIME 類型
- 自動 404 錯誤頁面

---

## 🐛 常見問題

### Q: 伺服器無法啟動？
**A:** 確保您的連接埠 8000 未被其他應用佔用。如需更改連接埠，編輯 `server.py` 或 `server.js` 中的 `PORT` 變數。

### Q: 如何更改連接埠？
**A:** 
- Python: 編輯 `server.py` 最後一行 `run_server(8000)` 改為其他埠號
- Node.js: 編輯 `server.js` 第 13 行 `const PORT = 8000` 改為其他埠號

### Q: 如何將網站分享給他人？
**A:** 部署到線上（如 GitHub Pages、Netlify、Vercel）。
