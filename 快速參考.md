# 🚀 快速參考 - 三種使用方式

---

## 方式 1️⃣ 本地測試（立即使用）⚡

**適合**: 自己在電腦上測試

### Windows
```
雙擊 → 啟動伺服器.bat
```

### Mac / Linux
```bash
python3 server.py
```

**網址**: `http://localhost:8000`

---

## 方式 2️⃣ 本地打開（最簡單）📁

**適合**: 沒有網路環境、臨時展示

**直接打開**: `index.html` 用瀏覽器開啟即可

⚠️ **注意**: 購物車功能需要伺服器支持，本地打開可能無法完全運作

---

## 方式 3️⃣ 線上部署（最佳）🌐

**適合**: 正式上線、分享給別人、全球訪問

### 5 分鐘快速部署

```
1. 建立 GitHub 帳戶 (github.com)
   
2. 建立新 Repository
   名稱: pengmei-dumpling
   選擇: Public
   ✓ Add README
   
3. 上傳所有 .html, .css, .js 檔案
   
4. Settings → Pages → 選擇 main 分支 → Save
   
5. 等待 1-2 分鐘，獲得網址！
```

**網址範例**: `https://username.github.io/pengmei-dumpling`
