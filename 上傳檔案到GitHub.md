# 📤 在 GitHub 上傳檔案的完整步驟

## 第一次：建立 Repository 並上傳檔案

### 1️⃣ 登入 GitHub
```
訪問 github.com
輸入用戶名和密碼登入
```

### 2️⃣ 建立新 Repository
```
點擊頁面右上角 「+」 符號
   ↓
選擇 「New repository」
   ↓
出現新頁面
```

### 3️⃣ 填寫 Repository 資訊
```
Repository name: pengmei-dumpling
（或任意名稱）

Description: 膨嬤水餃 & 小帥甜點店官網
（可選）

選擇 「Public」（公開）
   ✓ 這樣所有人都能訪問

✓ 勾選 「Add a README file」

最後點擊 「Create repository」
```

### 4️⃣ 進入 Repository（建立成功後自動進入）
```
頁面上會顯示：
[你的用戶名] / pengmei-dumpling

中間有綠色按鈕 「Add file」
```

### 5️⃣ 上傳檔案 ⬅️ 這是您要找的位置！
```
點擊 「Add file」按鈕
   ↓
選擇 「Upload files」
   ↓
出現上傳區域
```

### 6️⃣ 選擇檔案
在上傳區域：
- 可以 **拖放** 檔案進去，或
- 點擊 **「choose your files」** 選擇檔案

**需要上傳的檔案**：
```
✓ index.html
✓ shop.html
✓ product.html
✓ cart.html
✓ checkout.html
✓ style.css
✓ products.js
✓ cart.js
✓ main.js
```

### 7️⃣ 提交上傳
```
填寫 Commit message：
「Upload website files」

點擊 「Commit changes」

✅ 完成！檔案已上傳
```

## 🎯 快速導覽

### 如果您已經有 Repository，要上傳更多檔案：

```
1. 進入您的 Repository 頁面
2. 點擊 「Add file」(綠色按鈕)
3. 選擇 「Upload files」
4. 拖放或選擇檔案
5. Commit changes
```

### Repository 頁面在哪？

```
方式 1：
GitHub 左邊 sidebar → 「Your repositories」
  ↓
點擊要上傳的 Repository 名稱

方式 2：
GitHub 右上角 → 用戶頭像 → 「Your repositories」
  ↓
點擊要上傳的 Repository 名稱

方式 3：
直接訪問 URL：
https://github.com/[你的用戶名]/pengmei-dumpling
```

## 💾 一次上傳多個檔案

**重要**: 您可以一次選擇多個檔案！

```
步驟：
1. 點擊 「Add file」→ 「Upload files」
2. 點擊 「choose your files」
3. 在檔案選擇對話框中：
   - 選擇第一個檔案
   - 按 Ctrl (Windows) / Cmd (Mac)
   - 點擊其他檔案
   - 一次全選 (Ctrl+A)
4. 打開所有檔案
5. 點擊 「Commit changes」
```

✅ **建議**: 一次上傳所有 9 個檔案，這樣最快！

## 🚀 啟用 GitHub Pages

上傳檔案後，還需要啟用 Pages 功能：

```
1. 進入 Repository
2. 點擊 「Settings」(齒輪圖標)
3. 左邊菜單選擇 「Pages」
4. 「Source」選擇：
   Branch: main
   Folder: / (root)
5. 點擊 「Save」
6. 等待 1-2 分鐘

✅ 會出現綠色提示：
   Your site is live at: https://...
```
