# 🥟 膨嬤水餃 & 小帥甜點店 - 冷凍美食電商

## 📝 專案描述

這是一個完整的前端電商網站原型，販賣膨嬤的手工冷凍水餃和小帥設計的甜點。具備完整的購物流程（瀏覽→篩選→購物車→結帳），採用**溫馨日系文青風**設計美學。

## 🛒 核心功能

### 1️⃣ 首頁
- Hero 標語區 ("一口爆汁，尋回記憶中的家常美味")
- 三大賣點展示 (圖示卡片)
- 熱銷排行榜 (3 款推薦商品)
- 顧客好評區 (5 星評價)
- CTA 按鈕導向商城

### 2️⃣ 商品列表頁
- **分類篩選**: 經典 / 海鮮 / 蔬食 / 限定 / 蛋糕 / 巧克力 等
- **商品卡片**: 圖片、名稱、價格、簡述、快速加購
- **即時加購**: 直接在列表頁選擇數量加入購物車
- 13 款商品完整資訊

### 3️⃣ 商品詳情頁
- 商品圖片與多角度介紹
- 規格、成分、過敏原資訊
- 料理指南 (水煮/煎餃/蒸餃做法)
- 用戶真實評價與星等
- 數量選擇與加入購物車

### 4️⃣ 購物車頁面
- **購物車列表**: 商品、數量、小計、移除
- **免運進度條**: 視覺化距離免運還需多少
- **訂單摘要**: 商品小計 + 運費 + 稅金 + 應付總額
- **混搭組合建議**: 協助達到免運門檻
- **繼續購物**按鈕

### 5️⃣ 結帳頁面
- **訂購人資訊**: 姓名、郵箱、手機、地址
- **配送方式**: 黑貓宅急便 / 超商冷凍超取
- **支付方式**: 信用卡 / 銀行轉帳 / 貨到付款
- **訂單確認**: 模擬訂單生成與下單成功
- **安心購物保障**: 安全付款 / 冷藏配送 / 7 天退貨

## 💳 電商特色

### 購物車持久化
- 使用 `LocalStorage` 儲存購物車
- 刷新頁面、關閉瀏覽器都不會消失
- 支援多品種、多數量

### 智能運費計算
- **免運門檻**: NT$1,200
- **冷凍運費**: NT$80
- **自動計算**: 達到門檻時自動免運
- **視覺進度**: 購物車頁面顯示進度條

### 價格計算邏輯
```
應付金額 = 商品小計 + 運費 + 稅金(5%)
```

## 🚀 使用方式

### 1. 開啟網站
直接在瀏覽器中打開 **`index.html`** 即可開始瀏覽。

```bash
# Windows 使用者
start "c:\Users\Dr. Chang\Desktop\wedsite design\index.html"

# macOS 使用者
open /path/to/index.html

# Linux 使用者
xdg-open /path/to/index.html
```

### 2. 完整購物流程
1. **首頁** → 查看推薦商品與好評
2. **商品列表** → 按分類篩選或直接加購
3. **購物車** → 檢查訂單、查看免運進度
4. **結帳** → 填寫資訊、選擇配送與支付方式
5. **訂單成立** → 顯示訂單編號

### 3. 購物車同步
- 在商品列表頁加購 → 購物車徽章自動更新
- 在商品詳情頁加購 → 購物車數量累加
- 刷新頁面 → 購物車內容保存

## 💡 功能實現細節

### 購物車邏輯 (`cart.js`)
```javascript
// 加入購物車
Cart.addToCart(productId, quantity);

// 計算總額
Cart.getTotal(); // 商品小計 + 運費 + 稅金

// 取得購物車項目
Cart.getCart();

// 清空購物車
Cart.clearCart();
```
